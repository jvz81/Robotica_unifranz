Procedimiento para clonar repositorio en git bash:
    Obtener URL del repositorio en la opción CODE
    cambiar al directorio donde se desea clonar el repositorio (cd)
    git clone  "dirección URL del repositorio"

Para actualizar el repositorio local lo mas facil es:
    git pull

Procedimiento para actualizar repositorio:
    1. git status   Para verificar estado de los archivos para actualizar 
    2. git add .    Para subir todos los archivos
    3. git commit -m "texto"    Mensaje para subir
    4. git -push

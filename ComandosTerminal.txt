Lista de comandos utilizados en terminal o consola
    pwd     	Indica la ubicacion de la carpeta en donde se 
            	encuentra el terminal
    cd      	Cambiar directorio
    ls      	Lista de archivos y carpetas
    la		    lista todo, incluso archivos ocultos
    mkdir   	Para crear directorio 
    touch   	Para crear archivos
    nano    	Abre un editor de texto en la consola
    cat     	Imprime el contenido de un archivo en la terminal
    source  	Actualiza la terminal para utilizar el o alias codigo de un archivo .bashrc
    
    Para abrir el administrador de archivos en modo superusuario (sudo nautilus)
    Para editar archivo con super usuario (sudo gedit <archivo>)
